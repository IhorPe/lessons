Иконки

Небольшие простые изображения, которые усиливают эффект текста или самостоятельно несут некоторую, определённую информацию. В большинстве случаев это векторные изображения (SVG, PDF). Иконки могут интегрироваться через CSS (background и др.)

Так же и через тег IMG в HTML файле. Как пример контентные иконки, это могут быть логотипы компаний и тд.

Элементы дизайна

Векторные или растровые изображения (возможно с прозрачностью), которые являются частью дизайна. Элементы дизайна, чаще всего, интегрируются через CSS ( background и др.)

Контентные изображения

Изображения которые и есть контентом сайта. Чаще это растровые изображения (фотографии).(JPG/PNG -> WEBP). Изображения могут изменятся, через админпанель, мы их интегрируем через тег IMG в HTML файл. PNG формат изображения с α-каналом (rgba), с прозрачностью. Файл PNG больше чем JPG (тяжелее, больше mb). Есть ещё формат изображения WEBP, тоже с α-каналом (rgba), он меньше по объёму чем JPG/PNG.
Сначала мы оптимизируем изображение на ресурсе TinyPNG.com. Затем конвертируем его в WEBP на сайте Convertio.co.
